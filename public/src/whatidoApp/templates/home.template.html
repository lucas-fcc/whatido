How It Works?

Create a Task List

Create a list to group the things you need to do, be it your shopping list, your schoolwork, etc. You can have as many lists as you want.

Add Tasks

Add to the task list you created before the tasks that belong to it, so you will keep them organized.

Ask a Suggestion

After you add your tasks to a task list, if you feel uncertain by which one start, ask for a suggestion.

Finish Tasks

The tasks that you are finishing can be marked as completed and then they will no longer be drawn as a suggestion.
